Mirador-based IIIF viewer with zoom to a rectangle area on canvas specfied by argument.
If you want to acquire the xywh cordinates on canvas, please use https://github.com/yamadashoji18/iiif_xywh_tool.

Usage:
https://servername/path/iiif-viewer.html?manifest=manifesturl&canvas=canvasnumber-1&xywh=x:y:w:h

Example:
https://www.nichibun.ac.jp/YoukaiGazou/iiif-viewer.html?manifest=https://iiif.nichibun.ac.jp/IIIF/manifest/EMK/BB10004978.json&canvas=0&xywh=42482:214:3551:3252
